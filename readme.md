Usage
---

Use it with [Vundle](http://github.com/gmarik/Vundle.vim)

Edit vimrc

```vim
Plugin 'chunpu/vim-monokai'

set t_Co=256
colorscheme monokai
```

Remember to run `vim +PluginInstall +qall` in shell after add Plugin to vimrc
